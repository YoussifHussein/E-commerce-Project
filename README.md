# 🛒 E-Commerce Project

A modern and scalable E-Commerce application designed to simulate real-world online shopping platforms with complete product management, user interaction, and order processing workflows.

# 📌 Project Overview

This project demonstrates the development of a complete E-Commerce system with features commonly used in real online stores.

The application focuses on:

- Product management
- User authentication
- Shopping cart functionality
- Order processing
- Data handling and scalability
- Responsive user experience

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YoussifHussein/E-commerce-Project.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd E-commerce-Project
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

or

```bash
npm install
```

---

## 4️⃣ Run the Project

### Backend

```bash
python app.py
```

or

```bash
npm start
```

### Frontend

Open:

```bash
index.html
```

or run React app:

```bash
npm run dev
```
